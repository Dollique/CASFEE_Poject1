### Prerequisites

You need these:

```
* Yarn
* ESLint
* NodeJS > 12.x
```

Info: This project uses ES6 and ESM for JS and NodeJS.
Babel is not yet set up.

### Installing dependencies

Frontend
```
cd .\root\frontend\
yarn
```

Backend
```
cd .\root\backend\
yarn
```

## Running NodeJS

```
cd .\root\backend\
yarn run start
```
will run nodemon
